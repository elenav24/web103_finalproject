# User Stories

## Outline User Roles

- **Gift Giver**: a user who manages a personal list of contacts and plans gifts for the people in their life
- **Guest**: a visitor who has not yet created an account

## Draft User Stories

1. As a gift giver, I want to add a person to my contacts with their name and relationship to me, so that I can keep track of everyone I buy gifts for in one place.
2. As a gift giver, I want to attach a birthday to a contact, so that I know when their birthday is coming up without having to remember it myself.
3. As a gift giver, I want to add a custom event (like Christmas or an anniversary) to a contact's profile, so that I can track gift occasions that go beyond just birthdays.
4. As a gift giver, I want to add gift ideas to a contact's profile with a name, description, and estimated price, so that I always have options ready when an occasion approaches.
5. As a gift giver, I want to attach a link to a gift idea, so that I can easily find and purchase it when I'm ready.
6. As a gift giver, I want to set a spending budget for a contact or event, so that I can plan my gift ideas without going over what I can afford.
7. As a gift giver, I want to see how the total estimated cost of my gift ideas compares to my budget, so that I can adjust my plans before I overspend.
8. As a gift giver, I want to mark a gift idea as "purchased" or "given", so that I can track my progress and avoid accidentally buying the same thing twice.
9. As a gift giver, I want to see a list of upcoming gift deadlines sorted by how soon they are, so that I know which occasions need my attention first.
10. As a gift giver, I want to edit or remove a contact, so that I can keep my list accurate as relationships and circumstances change.
11. As a gift giver, I want to edit or delete a gift idea, so that I can update my plans as I learn more about what someone wants.
12. As a gift giver, I want to search my contacts by name, so that I can quickly pull up someone's profile without scrolling through my whole list.
13. As a guest, I want to read about what GiftGiver does on the landing page, so that I can decide if it's worth signing up.
14. As a guest, I want to create an account with an email and password, so that I can start building my own gift lists.
